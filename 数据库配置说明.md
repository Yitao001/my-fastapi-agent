# 数据库配置说明

本项目支持灵活的数据库表结构配置。

## 配置文件位置
配置文件位于：`config/database.yml`

## 配置项说明

### 1. MySQL连接配置
```yaml
mysql:
  host: "数据库主机地址"
  user: "数据库用户名"
  password: "数据库密码"
  database: "数据库名称"
```

### 2. 表结构配置
```yaml
table_structure:
  table_name: "谈话记录表名"
  fields:
    student_id: "学生ID字段名"
    content: "谈话内容字段名"
    created_time: "创建时间字段名"
  query_limit: 100
```

## 使用步骤

1. 根据您的数据库表结构，修改 `config/database.yml` 中的配置
2. 启动API服务：`python api.py`
3. 调用API接口，传入学生ID即可获取人物画像

## API调用示例

```bash
curl -X POST http://localhost:8000/analyze \
  -H "Content-Type: application/json" \
  -d '{"participant_id": "S2023003"}'
```

## 注意事项

1. 确保数据库表中至少包含三个字段：学生ID、谈话内容、创建时间
2. `query_limit` 用于限制每次查询返回的记录数量，避免查询过多数据影响性能
3. 如果字段名有特殊字符（如空格、连字符等），可能需要使用反引号或方括号进行转义（根据数据库类型）
