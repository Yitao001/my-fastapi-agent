## 2. 简单更换：通义千问不同版本

如果您只是想换通义千问的不同版本，**只需修改一个文件**！

### 步骤1：编辑配置文件

打开 `config/rag.yml`，修改 `chat_model_name`：

```yaml
# 当前配置
chat_model_name: qwen3-max

# 可选值：
# qwen-turbo    - 最快，成本最低
# qwen-plus     - 平衡
# qwen-max      - 能力强
# qwen3-max     - 最新最强（当前）
```

### 步骤2：重启服务

保存文件后，重启API服务即可生效。

## 3. 高级更换：切换到其他模型

如果您想换成 OpenAI GPT、Claude 等其他模型，需要以下步骤：

### 方案A：使用多模型工厂（推荐）

我已经为您创建了一个支持多模型的版本！

#### 步骤1：备份原文件

```powershell
copy model\factory.py model\factory.py.backup
copy config\rag.yml config\rag.yml.backup
```

#### 步骤2：替换文件

```powershell
copy model\factory_multi.py model\factory.py
copy config\rag_example.yml config\rag.yml
```

#### 步骤3：配置您想用的模型

编辑 `config/rag.yml`，选择模型提供商：

**示例1：使用 OpenAI GPT**
```yaml
model_provider: openai

openai_api_key: sk-您的API密钥
openai_base_url: https://api.openai.com/v1
chat_model_name: gpt-4o
embedding_model_name: text-embedding-3-small
```

**示例2：使用 Claude**
```yaml
model_provider: anthropic

anthropic_api_key: sk-ant-您的API密钥
chat_model_name: claude-3-5-sonnet-20241022
```

**示例3：继续使用通义千问**
```yaml
model_provider: tongyi

chat_model_name: qwen3-max
embedding_model_name: text-embedding-v4
```

#### 步骤4：安装对应依赖

根据您选择的模型，安装需要的包：

```powershell
# OpenAI
pip install langchain-openai

# Anthropic Claude
pip install langchain-anthropic
```

#### 步骤5：重启服务

---

### 方案B：手动修改（不推荐，适合学习）

如果您想手动修改原文件，按以下步骤：

#### 步骤1：修改 `model/factory.py`

将原内容：
```python
from langchain_community.chat_models.tongyi import ChatTongyi
from langchain_community.embeddings import DashScopeEmbeddings

class ChatModelFactory(BaseModelFactory):
    def generator(self):
        return ChatTongyi(model=rag_conf["chat_model_name"])

class EmbeddingsFactory(BaseModelFactory):
    def generator(self):
        return DashScopeEmbeddings(model=rag_conf["embedding_model_name"])
```

替换为（以OpenAI为例）：
```python
from langchain_openai import ChatOpenAI, OpenAIEmbeddings

class ChatModelFactory(BaseModelFactory):
    def generator(self):
        return ChatOpenAI(
            model="gpt-4o",
            api_key="您的API密钥"
        )

class EmbeddingsFactory(BaseModelFactory):
    def generator(self):
        return OpenAIEmbeddings(
            model="text-embedding-3-small",
            api_key="您的API密钥"
        )
```

#### 步骤2：安装依赖

```powershell
pip install langchain-openai
```

#### 步骤3：重启服务

## 5. 常见问题

### Q1: 更换模型后需要重新训练吗？

**A:** 不需要！人物画像分析是直接调用模型，不需要训练。

### Q2: 向量库需要重建吗？

**A:** 如果更换了嵌入模型（embedding model），建议删除 `chroma_db` 目录重新生成。

### Q3: API Key 放哪里安全？

**A:** 可以放在 `.env` 文件里，然后通过环境变量读取，不要提交到 Git。

### Q4: 可以同时用多个模型吗？

**A:** 可以！但需要修改代码，让不同的功能用不同的模型。
