### 2. ✅ P1: LLM调用重试机制

**新增文件**：`utils/retry_utils.py`

**功能特性**：
- 指数退避重试策略
- 可配置重试次数（默认3次）
- 可配置初始延迟（默认2秒）
- 可配置最大延迟（默认10秒）
- 详细的重试日志记录

**配置示例**：
```python
llm_retry_config = RetryConfig(
    max_retries=3,
    initial_delay=2.0,
    max_delay=10.0,
    exponential_base=2.0,
    timeout=60.0
)
```

**使用方式**：
```python
@with_retry(config=llm_retry_config)
def _generate_persona(...):
    # LLM调用会自动重试
    ...
```
