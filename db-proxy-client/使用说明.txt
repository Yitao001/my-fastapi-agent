============================================================
数据库代理客户端 - 使用说明
============================================================

第一步：配置
------------
1. 复制 .env.example 为 .env
2. 用记事本打开 .env 文件，修改以下内容：

   # 中继服务地址（云服务器的 WebSocket 地址）
   RELAY_URL=ws://47.102.145.89:8002

   # 客户端ID（自己起个名字）
   CLIENT_ID=my_client_001

   # 数据库配置
   MYSQL_HOST=127.0.0.1
   MYSQL_PORT=3306
   MYSQL_USER=root
   MYSQL_PASSWORD=你的数据库密码
   MYSQL_DATABASE=student_talk_db

第二步：启动
------------
1. 安装依赖（首次需要）：
   pip install -r requirements.txt

2. 运行客户端：
   双击 start.bat
   或命令行：python client.py

第三步：测试
------------
看到 "已连接到中继服务" 就表示成功了！
